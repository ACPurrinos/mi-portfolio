SOBRE MÍ

Historia

Mi camino profesional comenzó en áreas administrativas y operativas, donde desarrollé una visión integral de los procesos de negocio. En paralelo, estudié psicopedagogía, lo que me permitió comprender los procesos de aprendizaje y cómo potenciar habilidades en otras personas.

Esta combinación fortaleció mi capacidad para resolver problemas y me llevó a descubrir la programación como vocación. Me interesan especialmente el desarrollo backend y la creación de aplicaciones escalables y seguras.

Estudios

Actualmente curso la Tecnicatura en Desarrollo de Software, con el objetivo de profundizar en programación backend, arquitectura de software y buenas prácticas para construir aplicaciones seguras y escalables. Paralelamente, completé cursos de Java (Educación IT, Codo a Codo, Oracle Next Education), desarrollo web (Henry) y fundamentos de computación en la nube (AWS Practitioner).

Mantengo una formación continua en nuevas tecnologías, convencida de que la innovación y el crecimiento profesional son constantes.

Objetivos

Como Desarrolladora de Software Junior, mi objetivo es crear aplicaciones backend seguras, eficientes y escalables.

Mi formación previa y experiencia en administración me brindan una sólida capacidad para trabajar en equipo, comunicarme de forma efectiva y comprender necesidades del negocio. Estas habilidades complementan mi perfil técnico y me permiten integrarme con facilidad en entornos colaborativos.

MIS PROYECTOS

Book Finder

Portal de venta de libros con carrito de compras, desarrollado colaborativamente en el bootcamp de Henry. FrontEnd realizado con React y Material UI. Incluye integraciones como MercadoPago y Auth0, filtros de búsqueda, paginado y autenticación segura.

CRUD para administración

Panel de administración de Book Finder con acceso restringido. Su BackEnd fue desarrollado con tecnologías como Node.js, Express, Sequelize y Postgres, garantizando un manejo eficiente de los datos. Además, utiliza Cloudinary para el almacenamiento de imágenes.

Literalura

Aplicación Java basada en Spring Boot que permite buscar y gestionar libros y autores utilizando una API externa (Gutendex). La aplicación utiliza una base de datos relacional para almacenar los libros y sus autores.

Encriptador Challenge

Ejercicio de maquetado en HTML5 y CSS. La web incluye funciones en Javascript para encriptar y desencriptar textos ingresados por el usuario

Mi CV Web

Front Moovies

Proyecto final para la materia Frontend en el marco de la carrera Tecnicatura Superior en Desarrollo de Software. El mismo consiste en una aplicacón de streaming de películas y series realizada con HTML, CSS y JavaScript consumiendo la API TheMoviedb.org. También se utilizó mockApi para simular la gestión de autenticación de usuarios y guardar series y películas favoritas en el perfil del usuario.
