SOBRE MÍ

Quién soy

Soy Andrea, una Desarrolladora de Software Junior. Tras varios años de experiencia en administración y estudios en psicopedagogía, adquirí habilidades valiosas en análisis, planificación y resolución de problemas complejos. Estas competencias, combinadas con mi pasión por la tecnología, me llevaron a descubrir la programación como mi verdadera vocación.

Busco especializarme en el desarrollo backend, creando aplicaciones escalables y seguras, y sigo aprendiendo constantemente nuevas herramientas y lenguajes que potencien mis proyectos.

Qué estudio

Actualmente estoy cursando la Tecnicatura en Desarrollo de Software, donde mi meta es profundizar en programación backend y buenas prácticas para construir aplicaciones seguras y escalables. Además, he completado varios cursos de Java (Educación IT, Codo a Codo) y desarrollo web (Henry), fortaleciendo mis conocimientos en bases de datos, APIs y arquitectura de software.

Como interés personal, en un futuro me gustaría incursionar en el área de IA y explorar cómo se relaciona con la educación y las neurociencias, mis otras pasiones.

Qué hago

Como Desarrolladora de Software Junior, mi objetivo es construir aplicaciones backend seguras y escalables.

Gracias a mi formación anterior y experiencia en administración, aporto una excelente capacidad de trabajo en equipo, comunicación y comprensión de necesidades, cualidades que complementan mis habilidades técnicas y me permiten integrarme eficazmente en proyectos colaborativos.

MIS PROYECTOS

Book Finder

Web de venta de Libros con carrito de compras. FrontEnd realizado con React y Material UI. Incluye integraciones como MercadoPago y Auth0, filtros para búsquedas de productos, paginado y JWT.

CRUD para administración

Es el panel de administración de Book Finder, con acceso restringido. Al igual que el portal de ventas, tiene un BackEnd realizado con tecnologías como Sequelize, Postgres, Node Js y Express. Utiliza servicios de Cloudinary para almacenamiento de imágenes.

Literalura

Literalura. Aplicación Java basada en Spring Boot que permite buscar y gestionar libros y autores utilizando una API externa (Gutendex). La aplicación utiliza una base de datos relacional para almacenar los libros y sus autores.

Encriptador Challenge

Ejercicio de maquetado en HTML5 y CSS. La web incluye funciones en Javascript para encriptar y desencriptar textos ingresados por el usuario
